### How do I use this?
Here's a quick guide
1. Install python3 on your computer.
2. Open your computers terminal (bash, shell, ect.)
3. Run the command (this installs and the dependencies for the script)
` ` `
pip3 install -r requirements.txt
` ` `
4. Run the command
` ` `
python3 main.py
` ` `
5. Enjoy :)
